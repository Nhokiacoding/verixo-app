# Virtual Phone Number Marketplace

A React app for SMS verification services using virtual phone numbers.

## Setup

1. Ensure Node.js is installed (version 18+ recommended).
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open http://localhost:5173 in your browser.

## Features

- Home page with hero section, features, and navigation.
- Built with Vite and React.

## Build

Run `npm run build` to build for production.
